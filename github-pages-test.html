Implementation Code

Add this single line of code to any website to enable the feedback widget. Replace YOUR_API_KEY with your actual API key from the FeedbackFlow dashboard.

<!-- FeedbackFlow Widget --> <script src="https://YOUR_REPLIT_URL.replit.dev/widget.js" data-api-key="YOUR_API_KEY"></script>

Technical Specifications

  • Pure JavaScript (ES6+) - No dependencies required
  • Gzip compressed size: ~8KB (smaller than most images)
  • Compatible with all modern browsers (Chrome, Firefox, Safari, Edge)
  • HTTPS required for production use
  • CORS-enabled for cross-domain requests
  • Rate limiting: 100 requests per hour per API key
  • Real-time sentiment analysis powered by AI
  • Automatic retry logic with exponential backoff
  • GDPR and privacy-compliant data handling

What Happens When Customers Submit Feedback?

When someone uses your feedback widget, here's the complete flow:

  1. Instant Submission: Feedback is securely sent to FeedbackFlow servers
  2. AI Analysis: Automatic sentiment analysis (positive, negative, neutral)
  3. Real-time Processing: Data appears instantly in your dashboard
  4. Context Capture: Page URL, referrer, and timestamp automatically recorded
  5. Customer Confirmation: Success message displayed to the user
  6. Dashboard Update: Analytics and trends updated in real-time
